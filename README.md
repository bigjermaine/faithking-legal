## Enable GitHub Pages

1. Open **Settings → Pages**
2. **Source:** Deploy from a branch
3. **Branch:** `main` · **Folder:** `/ (root)`
4. Save and wait ~1 minute

This repository must stay **public** so App Store reviewers can open the links.
